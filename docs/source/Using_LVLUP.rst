Using_LVLUP
=============

This page provides instructions on how to use LVLUP


Installation
------------
repository can be installed into an environemnt using pip, as per below. (if applicable)

.. code-block:: bash

	pip install git+https://github.com/Ellutze/LVLUP.git
	
or with specific version:

.. code-block:: bash

	pip install git+https://github.com/Ellutze/LVLUP.git@v0.8.2
